# Polynomial Server 0.2.4

## 围棋棋局删除

- “进行中的棋局”新增删除按钮。
- “最近棋谱”新增删除按钮。
- 删除进行中的棋局前会明确提示棋局将从双方列表永久移除。
- 删除最近棋谱前会明确提示棋谱将从双方记录永久移除。
- 只有棋局参与者或管理员可以删除棋局，其他用户不能通过接口越权删除。
- 删除棋局时同步清理关联的已接受邀请记录。
## 用户主页与战绩

- 围棋中心新增每位用户的个人围棋主页，并与棋盘偏好合并展示。
- 单机人机对弈与多人正式对局分别统计局数、胜负、和棋与胜率。
- 点击棋手姓名可查看其公开/可访问棋谱，以及当前用户与该棋手的交手胜率。
- 研究棋盘、共享棋盘与无效局不计入竞技胜率。
