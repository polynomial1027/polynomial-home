# 0.2.0 Demo 更新记录

发布日期：2026-07-20

## 新增

- 围棋中心：自由研究、题库、人机对弈、账号邀请联机、共享棋盘。
- 围棋规则引擎：提子、禁入、简单劫、全局同形禁着、让子、双停与中国/日本基础计分。
- 正式棋局设置：颜色/随机、让子、贴目、计时、读秒、悔棋和棋谱范围。
- 双方数目确认与自动结果；不存在管理员实时仲裁入口。
- KataGo 常驻 GTP 适配器和后台可编辑难度档。
- 棋局记录、终局与步骤查看、SGF 导出；研究树保存和 SGF 导入。
- 围棋细粒度账号权限和后台题库编辑器。
- 全站统一问题反馈和后台汇总工作流。
- 可选 `POLYNOMIAL_DATA_DIR`，方便代码与生产数据分离。
- 提供全新服务器构建隔离 JupyterLab 4.6.1 镜像的 Dockerfile。

## 兼容与迁移

- 读取 0.1.5 `store.json` 时自动补齐新设置、数组、权限、0.2.0 公告和示例题。
- 原聊天、网盘、课程、作业、Notebook、消消乐数据保持原结构。
- Nginx 必须新增 `/go-socket` WebSocket 代理。

- `npm test`：14 项规则和服务测试通过。
- `npm audit --omit=dev`：0 个已知漏洞，发布依赖无过期版本。
- 完整冒烟：页面 200、双账号登录、邀请/接受、双停、双方数目、自动结束、SGF、反馈后台、`/go-socket` 均通过。
